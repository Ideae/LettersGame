INSTRUCTIONS:

This project was built in Unity version 5.2.1f1.

If you would like to run the application, double click 'A5-part1.exe' in this folder.
Note: If you decide to move the .exe to another folder, you must also move the A5-part1_Data
folder with it.

If you would like to view the project within Unity, double click the '.unitypackage' file
in this folder to import the project into Unity. You can then view the source files as well.
